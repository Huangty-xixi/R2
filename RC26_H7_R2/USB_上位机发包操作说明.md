# USB上位机发包操作说明

本文档用于说明当前固件版本下，上位机如何通过 USB CDC 向下位机发送控制帧。

## 1. 帧格式

- 帧头: `0xAA 0x55`
- 数据区: 20 字节 (`data[0] ~ data[19]`)
- 校验: 1 字节
- 帧尾: `0x0D 0x0A`

总长度: 25 字节

完整格式:

`AA 55 + data[20] + checksum + 0D 0A`

校验算法:

`checksum = (data[0] + data[1] + ... + data[19]) & 0xFF`

## 2. 数据区总定义

当前主控只使用前 3 个数据字节，其余预留:

- `data[0]`: 模式选择字节
- `data[1]`: 模式动作字节0
- `data[2]`: 模式动作字节1
- `data[3] ~ data[19]`: 预留，建议填 `0x00`

## 3. 模式选择 (`data[0]`)

一次只选一个模式位:

- bit0: 底盘模式 (`master_chassis_mode`)
- bit1: 武器模式 (`master_weapon_mode`)
- bit2: 抬升模式 (`master_lift_mode`)
- bit3: KFS模式 (`master_kfs_mode`)

若多个位同时为 1，固件按如下优先级处理:

底盘 > 武器 > 抬升 > KFS

## 4. 各模式动作定义

## 4.1 底盘模式 (bit0=1)

使用:

- `data[1]` = 底盘动作字节0
- `data[2]` = 底盘动作字节1 (当前预留)

`data[1]` 位定义:

- bit7~6: 速度档
  - `01` 低速 (最大速度 1/3)
  - `10` 中速 (最大速度 2/3)
  - `11` 高速 (最大速度 1)
- bit5~3: 平移方向
  - `001` 前
  - `010` 后
  - `011` 左
  - `100` 右
- bit2~1: 旋转方向
  - `00` 不旋转
  - `01` 左旋
  - `10` 右旋
- bit0: 活动电机
  - `1` 伸出
  - `0` 收回

## 4.2 武器模式 (bit1=1)

使用:

- `data[1]` = `master_weapon_action_bits`

位定义:
    (0关1开)
- bit0: 舵机
- bit1: 夹爪
- bit2: 吸盘1
- bit3: 吸盘2
- bit4: 吸盘3
- bit5: 吸盘4

## 4.3 抬升模式 (bit2=1)

使用:

- `data[1]` = `master_lift_action_bits`

位定义:

- bit0: 抬升方向 (`1` 上升, `0` 下降)
- bit1: 伸缩方向 (按当前固件映射)（1伸出 0收回）
- bit2~bit7: 预留

## 4.4 KFS模式 (bit3=1)

使用:

- `data[1]` = `master_kfs_action_bits_0`
- `data[2]` = `master_kfs_action_bits_1`

`data[1]` 位定义:

- bit0~1: 三档旋转
  - `00` -> p1
  - `01` -> p2
  - `10` -> p3
  - `11` 预留
- bit2: 前臂两档
- bit3~5: 主轴抬升状态
  - `001` -> 状态1
  - `010` -> 状态2
  - `011` -> 状态3
  - `100` -> 状态4
  - 其他值预留
- bit7~6: 伸缩电机A(above)命令
  - `00` 停止
  - `01` 伸出 (速度 -100)
  - `10` 收回 (速度 +100)
  - `11` 预留

`data[2]` 位定义:

- bit1~0: 伸缩电机B(below)命令
  - `00` 停止
  - `01` 伸出 (速度 -100)
  - `10` 收回 (速度 +100)
  - `11` 预留
- bit7~2: 预留

## 5. 发送周期建议

- 推荐: 10ms/帧 (100Hz)
- 若只做低频离散动作，20ms/帧也可

## 6. 示例

示例: 进入武器模式，动作字节 `0x41`，其他为 0

- `data[0] = 0x02` (武器模式)
- `data[1] = 0x41`
- `data[2..19] = 0x00`

完整帧:

`AA 55 02 41 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 43 0D 0A`

说明:

- `0x43 = 0x02 + 0x41` (其余数据均为 0)
